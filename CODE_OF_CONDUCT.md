# Code of Conduct

## Our Standards

Be respectful, constructive, and professional. We're building tools, not drama.

**Expected behavior:**

- Constructive feedback with specific suggestions
- Respecting different viewpoints and experience levels
- Focusing on what's best for the project
- Being direct and honest without being hostile

**Unacceptable behavior:**

- Personal attacks or insults
- Harassment of any kind
- Publishing others' private information
- Trolling or deliberately inflammatory comments

## Enforcement

Project maintainers will remove, edit, or reject contributions that violate this code of conduct. Repeated violations result in a ban.

## Reporting

Report issues to conduct@gladlabs.io. All reports will be reviewed and handled confidentially.
