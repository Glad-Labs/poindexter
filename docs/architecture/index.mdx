Poindexter is modeled after human brain anatomy. Each region runs as an
independent process or container and the parts communicate exclusively through
shared PostgreSQL tables — no direct imports across region boundaries. The
result is a system you can monitor, restart, and reason about region by region.

For the full mastery-grade walk-through (regions, data flow, technology
choices, principles, and the request lifecycle), start with the
[architecture overview](./overview). The cards below jump straight to the
topic you need.
