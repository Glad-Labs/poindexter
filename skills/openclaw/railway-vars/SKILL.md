# Railway Variables

View or update environment variables on Railway.

## Usage

```bash
scripts/run.sh                          # List all variables
scripts/run.sh get KEY_NAME             # Get specific variable
scripts/run.sh set KEY_NAME VALUE       # Set a variable
```
