# Sprint Status

Shows the current sprint status from GitHub — open issues, milestones, and recent activity.

## Usage

```bash
scripts/run.sh                    # Full sprint overview
scripts/run.sh issues             # Open issues only
scripts/run.sh milestones         # Milestone summary
scripts/run.sh recent             # Recent closed issues (last 7 days)
```
